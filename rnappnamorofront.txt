When using react-native-progress to show image uploaded progress, it stuck on zero the percentage. To fix that, enter on this lib folder on node_modules and change the Circle.js file from this: {formatText(progressValue)} to this: {progress ? formatText(progress._value) : 0}

Troubleshooting
*******************************
ERROR: ".dex file cannot exceed 64K."
SOLUTION: Set 'multiDexEnabled true' into 'module/build.gradle'
EX: 'android {

    defaultConfig {
        ...

        // Enabling multidex support.
        multiDexEnabled true
    }
    ...
}'
-------------------------------
ERROR: "path not found on node_modules"
SOLUTION: "react-native start --reset-cache"
